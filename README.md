# app_subscription_flutter

Shared Flutter subscription package for iOS in-app subscriptions built on `flutter_inapp_purchase`.

## Install

```yaml
dependencies:
  app_subscription_core:
    git:
      url: https://github.com/cssdxc/app_subscription_flutter.git
      ref: main
```

## Exports

- `SubscriptionCoordinator`
- `SubscriptionCoordinatorConfig`
- `SubscriptionProductConfig`
- `SubscriptionActionResult`
- `IosSubscriptionHelper`
- `SubscriptionCatalog`
